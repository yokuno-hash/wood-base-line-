## 2. 進捗管理表 自然文クエリ機能

### コンセプト
LINEから「図面チェックできてない店舗ある？」のような自然文質問を、進捗管理表の列名にマッチさせて回答。

### 実装ポイント
- 別スプレッドシート対応（`PROGRESS_SPREADSHEET_ID` / `PROGRESS_SHEET_NAME_OVERRIDE` をScript Propertiesで指定）
- ヘッダー行自動検出（タイトル行があっても可）
- NFKC正規化で半角カナ `ﾊﾟｲｵﾆｱ` ⇔ 全角 `パイオニア` 同一視
- ファジーマッチで列名対応：図面/図面確認/図面チェック → 図面チェック など
- 「発注」→ 制作発注 + 発注書 を両方対象（multi:true）
- 月フィルタ（`5月`/`今月`/`来月`、全角数字 `３月` も対応、Date型対応）
- 店舗名抽出からひらがな除外（「確認中の店舗」誤検出防止）

### 完了/未完了判定
- 完了：丸記号全種（○ 〇 ◯ ◎ ⚪ ⚫ ● ⬤ ⭕ Ⓞ）+ 丸/まる/マル/済/完了/ok/✓ 等
- 日付列（依頼日/納品日時/完工日）：値あり = 完了、空欄 = 未入力
- 未完了：上記以外（△ 確認中 × 未 保留 対応中 差戻し 要確認 ー 等）

### 発火条件の厳密化
- タスク登録動詞（お願い/しといて/送って 等）含む → 進捗質問にしない
- 明示キーワード（未完了/まとめて/全部/一覧） → 即発火
- フィルタ条件 + クエリ動詞（ある？/教えて/まだ/?）→ 発火

### 表示仕様
- 表示上限 30件、超過分は店舗名のみリスト
- 依頼日は結果に表示しない

### 質問例
| LINE入力 | 動作 |
|---|---|
| 図面チェックできてない店舗ある？ | 図面チェック未完了 |
| 5月の図面チェックまだのところ教えて | 5月施工×図面未完了 |
| 確認中の店舗だけ教えて | 状態「確認中」抽出 |
| 6月施工で未完了ある？ | 6月施工分すべて |
| 田中さん担当で未完了ある？ | 担当者絞り込み |

## 3. 工程管理表 → 見積書 月別シート自動生成

### コンセプト
LINEで「東邦家具 5月分の見積書作って」と送るだけで、工程管理表から対象月の見積書シート（例：`26-5`）を自動作成。

### 設計
- **コード直書きゼロ** — 「見積連携設定」シートに1行追加するだけで複数案件対応
- 設定シート10列：設定ID/表示名/工程管理表ID/工程シート名/見積書ID/テンプレートシート名/担当者表示/単価/有効/備考

### 処理フロー
1. メッセージから設定特定 → 対象年月抽出
2. 既存シート確認（上書き防止）
3. 工程管理表→月別フィルタ→物件グループ化
4. 単価で集計（修正/新規ロジック）
5. テンプレ「原図」を copyTo → 「26-5」にリネーム → テンプレ右隣に配置
6. カバー＋明細書込＋合計反映
7. 要確認行は「見積要確認」シートに記録

### 金額計算ルール（統一）
- **金額 = 製図時間 × 単価**（新規・修正とも共通）
- 修正：数量=時間 / 単位=H / 単価=設定単価（1H時は空欄） / 金額=時間×単価
- 新規：数量=枚数 / 単位=枚 / 単価=空欄 / 金額=時間×単価

### テンプレ構造（東邦家具）
- 行10：税込合計(C列) / 消費税(F列) / 合計＝小計(I列)
- 行12：ヘッダー（No=A 名称=B 数量=F 単位=G 単価=H 金額=I 備考=J）
- 行14〜41：1ページ目 物件集計（行43に下部小計）
- 行53〜84：2ページ目 明細（自動跨ぎ）
- 行92〜126：3ページ目 明細

## 6. Webhook署名検証ロジックの緩和（運用復旧）

### 課題
[P0]で追加した署名検証が厳しすぎて、`LINE_CHANNEL_SECRET` をScript Propertiesに登録しただけでLINE Botが全リクエストを拒否してしまい無反応に。

### 対応
`verifyLineWebhook` を以下の方針に変更：
- シークレット未設定 → 通過
- **シークレットあり ＋ URLに token/sig 無し → 通過（警告ログのみ）**
- シークレットあり ＋ URLに token あり → 厳密照合（不一致は拒否）
- シークレットあり ＋ URLに sig あり → HMAC照合（不一致は拒否）

これでBotの稼働を維持したまま、`?token=` を付けた呼び出しに対しては検証が効く。
完全に偽Webhookを弾くには Webhook URL末尾に `?token=<シークレット>` を追加。
