# WOODBASE LINE 秘書AIシステム 機能一覧

---

## コンセプト

**メンバーは今まで通りLINEで会話するだけ。**  
Botが自動でタスク・スケジュールを管理します。

---


## できること

### 1. タスクの自動登録
LINEのメッセージからタスクを自動で読み取り、スプレッドシートに記録します。

**例：**
```
濱田さん 雨晴れクリニックの見積もりを4月25日までにお願いします
```
↓ 自動でスプレッドシートに登録

| 担当 | タスク | 期日 | ステータス |
|---|---|---|---|
| 濱田 | 見積もり提出 | 4/25 | 未着手 |

---

### 2. スケジュールの自動登録
日時・場所が含まれるメッセージをGoogleカレンダーに自動登録します。

**例：**
```
4月18日14時から雨晴れクリニックで現場確認です
```
↓ Googleカレンダーに自動登録

---

### 3. ファイル・画像の自動保存
グループに送られた写真・PDFをGoogleドライブに自動保存します。

```
WOODBASE/（Driveフォルダ）
├── 雨のち晴れクリニック/
│   ├── 20260414_現場写真.jpg
│   └── 20260414_図面.pdf
└── 未分類/
```

---

### 4. 自動リマインド（毎朝9時）
期日が近いタスクや未着手タスクをLINEグループに自動通知します。

| タイミング | 通知内容 |
|---|---|
| 期日3日前 | 「〇〇さん、期日が3日後です」 |
| 期日当日 | 「〇〇さん、本日期日のタスクがあります」 |
| 期日超過 | 「〇〇さん、期日を過ぎたタスクがあります」 |

---

### 5. 質問対応（秘書AI）
LINEで質問するとスプレッドシートを読んで即答します。

**例：**
```
雨晴れの残タスクは？
今週の予定教えて
```

---

### 6. 社内グループへの同時通知
外部グループ（施主・業者）でタスクが発生したとき、社内グループにも自動で通知します。

## v3.1 追加機能（2026-04-30）

### 7. 緊急度の自動判定 🚨

「至急」「急ぎ」「今日中」「最短」「GW前」などのキーワードを含むメッセージは自動で緊急フラグが付きます。

**例：**
```
至急！濱田さん 雨晴れの見積もりお願いします
```
↓
- 確認UI：📋 → 🚨 に変わる
- 担当者通知：「🚨【至急】あなたへのタスク」
- スプレッドシートの緊急度列に「高」と記録

**緊急キーワード一覧：**  
至急・急ぎ・急いで・ASAP・今すぐ・今日中・最短・緊急・GW前・GW中・明日まで・本日中

---

### 8. 連絡漏れ検出（毎朝9時）

タスクが登録されてから**3日経過しても未完了**の場合、自動でリマインドします。

| 通知先 | 内容 |
|---|---|
| 担当者のDM | 「このタスクの状況はいかがですか？」 |
| グループ | 「○○さんへ：対応状況を確認中です」 |

> 一度送ったタスクには再送しません（スパム防止）

---

### 9. 誤検出フィルター強化

以下のような「返事だけ」のメッセージはタスクとして登録されません。

| スキップされるパターン | 例 |
|---|---|
| 定型の承諾返事 | 了解しました・承知しました |
| お礼の言葉 | ありがとうございます |
| 挨拶 | お疲れ様でした・おはようございます |
| 確認報告 | 確認しました・見ました |

---

### 10. 重複タスク検出

同じグループ・同じ担当者・同じ内容のタスクが**48時間以内に登録済み**の場合、二重登録を自動でスキップします。

## 見積書 自動作成機能

工程管理表から、対象月の見積書シート（例：`26-5`）を自動で作成します。
工程管理表と見積書のペアを **「見積連携設定」シート** に登録するだけで、複数案件に対応できます。

### LINEからの使い方

| 入力例 | 動作 |
|---|---|
| `東邦家具 5月分の見積書作って` | 表示名で指定 |
| `toho 5月分 見積作成` | 設定IDで指定 |
| `〇〇家具 2026年6月の見積書作成` | 年も明示可 |
| `見積設定一覧` | 登録済み設定の一覧を返信 |

### 新しい案件を追加する手順

別の工程管理表＆見積書を追加する場合は、**管理用スプレッドシートの「見積連携設定」シートに1行追加するだけ**でOK。**コード変更は不要**です。

#### 1. 「見積連携設定」シートに1行追加

| 列 | 入力例 |
|---|---|
| A: 設定ID | `client_b`（半角英数の案件キー） |
| B: 表示名 | `〇〇家具` |
| C: 工程管理表ID | `1AbCdEf...`（その案件の工程管理表スプシURL内のID） |
| D: 工程シート名 | `工程管理表`（タブ名） |
| E: 見積書ID | `1XyZ...`（見積書スプシURL内のID） |
| F: テンプレートシート名 | `原図`（テンプレタブ名） |
| G: 担当者表示 | `担当者：〇〇様` |
| H: 単価 | `5000`（製図時間あたりの単価／空欄なら5000円） |
| I: 有効 | `TRUE` |
| J: 備考 | 任意メモ |

#### 2. スプシIDの取り出し方

スプシURLの `/d/` と `/edit` の間の **44文字** がID：
```
https://docs.google.com/spreadsheets/d/【ここがID】/edit#gid=0
```

#### 3. 共有設定

工程管理表と見積書の両方を **`woodbasegroup@gmail.com`** に**編集者**として共有してください。

### 見積書の記載ルール

#### 1ページ目（物件集計）
| 表示 | 内容 |
|---|---|
| 名称 | 【物件番号】物件名（新規〇枚／修正〇時間） |
| 数量 | 1 |
| 単位 | 式 |
| 単価 | 空欄 |
| 金額 | その物件の合計（製図時間×単価） |
| 備考 | 1行目：担当者表示／2行目以降：同上 |

#### 2ページ目以降（明細）

物件ごとに見出し行＋以下の明細：

| 分類 | 数量 | 単位 | 単価 | 金額 |
|---|---|---|---|---|
| 修正（時間=1） | 製図時間 | H | （空欄） | 製図時間×単価 |
| 修正（時間>1） | 製図時間 | H | 単価 | 製図時間×単価 |
| 新規 | 枚数 | 枚 | （空欄） | 製図時間×単価 |

### 注意点

- 月別シート名は `26-5` 形式（西暦下2桁+ハイフン+月、ゼロ埋めなし）
- **同名シートが既存の場合は上書きせず処理中止**してLINEで通知
- 新規シートは **テンプレートシートの右隣** に配置
- テンプレシート本体は触らずコピーして書き込み
- 自動計算できない行（製図時間空欄／検討のみ／WBF 等）は **「見積要確認」シート** に記録
