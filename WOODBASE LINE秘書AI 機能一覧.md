# WOODBASE LINE 秘書AIシステム 機能一覧

---

## コンセプト

**メンバーは今まで通りLINEで会話するだけ。**  
Botが自動でタスク・スケジュールを管理します。

---


## できること

### 1. タスクの自動登録
LINEのメッセージからタスクを自動で読み取り、スプレッドシートに記録します。

**例：**
```
濱田さん 雨晴れクリニックの見積もりを4月25日までにお願いします
```
↓ 自動でスプレッドシートに登録

| 担当 | タスク | 期日 | ステータス |
|---|---|---|---|
| 濱田 | 見積もり提出 | 4/25 | 未着手 |

---

### 2. スケジュールの自動登録
日時・場所が含まれるメッセージをGoogleカレンダーに自動登録します。

**例：**
```
4月18日14時から雨晴れクリニックで現場確認です
```
↓ Googleカレンダーに自動登録

---

### 3. ファイル・画像の自動保存
グループに送られた写真・PDFをGoogleドライブに自動保存します。

```
WOODBASE/（Driveフォルダ）
├── 雨のち晴れクリニック/
│   ├── 20260414_現場写真.jpg
│   └── 20260414_図面.pdf
└── 未分類/
```

---

### 4. 自動リマインド（毎朝9時）
期日が近いタスクや未着手タスクをLINEグループに自動通知します。

| タイミング | 通知内容 |
|---|---|
| 期日3日前 | 「〇〇さん、期日が3日後です」 |
| 期日当日 | 「〇〇さん、本日期日のタスクがあります」 |
| 期日超過 | 「〇〇さん、期日を過ぎたタスクがあります」 |

---

### 5. 質問対応（秘書AI）
LINEで質問するとスプレッドシートを読んで即答します。

**例：**
```
雨晴れの残タスクは？
今週の予定教えて
```

---

### 6. 社内グループへの同時通知
外部グループ（施主・業者）でタスクが発生したとき、社内グループにも自動で通知します。

**@520goeco** をLINEグループに招待するだけで自動的に動き始めます。

## 管理ツール

**スプレッドシート（4シート構成）**

| シート名 | 内容 |
|---|---|
| タスク管理 | 全タスクの一覧・ステータス管理・緊急度 |
| スケジュール管理 | 全予定の一覧 |
| メンバー管理 | 社内メンバー情報 |
| プロジェクト管理 | プロジェクト一覧・グループ紐付け |
| メッセージログ | 全グループの発言履歴 |
| 仮タスク | 確認待ちの一時保存 |
| Docs管理 | 案件サマリーのURL管理 |

## v3.1 追加機能（2026-04-30）

### 7. 緊急度の自動判定 🚨

「至急」「急ぎ」「今日中」「最短」「GW前」などのキーワードを含むメッセージは自動で緊急フラグが付きます。

**例：**
```
至急！濱田さん 雨晴れの見積もりお願いします
```
↓
- 確認UI：📋 → 🚨 に変わる
- 担当者通知：「🚨【至急】あなたへのタスク」
- スプレッドシートの緊急度列に「高」と記録

**緊急キーワード一覧：**  
至急・急ぎ・急いで・ASAP・今すぐ・今日中・最短・緊急・GW前・GW中・明日まで・本日中

---

### 8. 連絡漏れ検出（毎朝9時）

タスクが登録されてから**3日経過しても未完了**の場合、自動でリマインドします。

| 通知先 | 内容 |
|---|---|
| 担当者のDM | 「このタスクの状況はいかがですか？」 |
| グループ | 「○○さんへ：対応状況を確認中です」 |

> 一度送ったタスクには再送しません（スパム防止）

---

### 9. 誤検出フィルター強化

以下のような「返事だけ」のメッセージはタスクとして登録されません。

| スキップされるパターン | 例 |
|---|---|
| 定型の承諾返事 | 了解しました・承知しました |
| お礼の言葉 | ありがとうございます |
| 挨拶 | お疲れ様でした・おはようございます |
| 確認報告 | 確認しました・見ました |

---

### 10. 重複タスク検出

同じグループ・同じ担当者・同じ内容のタスクが**48時間以内に登録済み**の場合、二重登録を自動でスキップします。

## @Botへのメンションコマンド一覧

グループで `@Bot名` をメンションして使います。

| コマンド例 | 動作 |
|---|---|
| 残タスクは？ | 未完了タスクの一覧を表示 |
| 今週の予定は？ | スケジュール一覧を表示 |
| ○○案件まとめて | 会話履歴をAIで要約 |
| ○○案件 Docs更新 | Google DocsにAIサマリーを生成 |
| 完了しました | タスクを完了処理 |
